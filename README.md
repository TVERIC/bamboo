
# Bamboo

**Bamboo** is a *drop-in* replacement for Node.js. It is written in 100% JavaScript,
Bamboo does not have a single line written in C/C++.

Instead of running Node

```shell
node app.js
```

you can run Bamboo

```shell
bamboo app.js
```
